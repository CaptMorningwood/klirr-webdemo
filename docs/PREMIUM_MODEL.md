# Premium 2.0 model

Klirr Premium hjälper användaren att förbättra Budgeten, följa resultatet och låta Klirr hålla koll.

## Struktur
- **Förbättra min Budget:** `premiumValue` skapar en deterministisk personlig förbättringsplan med högst tre prioriterade möjligheter.
- **Följ min utveckling:** nuläges-snapshots jämför marginal, fasta utgifter och Budgethälsa. Importhistorik behandlas inte som utveckling.
- **Klirr håller koll:** automatisk koll, proaktiva insikter och reminders presenteras som lokal monitorering som körs när Klirr öppnas eller när användaren väljer Kör nu.
- **Mer kontroll:** flera Budgetar, delad Budget-demo och versioner finns kvar som sekundära verktyg.

## Estimat
Personliga estimat är konservativa, testbara och inte garantier. Duplicate income/review cleanup räknas som tillförlitlighet, inte garanterad marginal. Buffer kan stärka Budgeten utan att skapa fri extra marginal.

## Privacy
Lokal Premium-analys kräver inte AI. Premium Buddy+ går genom samma AI-samtycke och `src/lib/aiPrivacy.ts` som Budget Buddy och får inte skicka råa transaktioner.

## Free
Gratisprodukten förblir komplett: skapa/redigera Budget, Budgethälsa, Budget Checkup, import/granskning, basic Budget Buddy, actionkort, scenarier, export och cloud sync.
